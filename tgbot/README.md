# Telegram Bot для сайта

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram:
   - Отправьте команду `/newbot`
   - Укажите имя бота
   - Укажите username бота (должен заканчиваться на `bot`)
   - Скопируйте полученный токен

2. Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

3. Откройте `.env` и вставьте ваш токен:

```
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 3. Запуск бота

**Режим разработки** (с автоперезагрузкой):
```bash
npm run dev
```

**Продакшн режим**:
```bash
npm start
```

### 4. Запуск на хосте

Для продакшн запуска рекомендуется использовать PM2:

```bash
# Установка PM2 глобально
npm install -g pm2

# Запуск бота
pm2 start index.js --name telegram-bot

# Автозапуск при перезагрузке сервера
pm2 startup
pm2 save
```

## 📋 Функционал бота

### Интерфейс
- ✅ **Inline-кнопки** в сообщениях (не внизу экрана)
- ✅ **Фотография** в приветственном сообщении

### Главное меню
- 🛒 **Магазин** - каталог товаров (в разработке)
- 👤 **Кабинет** - личный кабинет пользователя
- ℹ️ **О НАС** - информация о компании
- ⚙️ **Админ-меню** - панель администратора (только для админа)

### Личный кабинет
- 📦 **Мои заказы** - просмотр заказов с сайта
- ❓ **Задать вопрос** - диалоговое окно:
  1. Запрос номера заказа
  2. Запрос вопроса
  3. Уведомление админа
- 👤 **Мой профиль** - информация о профиле

### Админ-панель
- 📊 **Статистика** - полная статистика бота
- 👥 **Пользователи** - (в разработке)
- 💬 **Сообщения** - (в разработке)
- 📎 **Файлы** - (в разработке)

## 🗄️ База данных

Бот использует две базы данных:
1. **bot.sqlite** - собственная БД бота с таблицами:
   - `bot_users` - пользователи бота
   - `bot_messages` - все сообщения
   - `bot_files` - загруженные файлы
   - `bot_orders_cache` - кеш заказов

2. **database.sqlite** - БД сайта (только чтение) для получения заказов

## ⚙️ Настройка

В файле `.env` доступны следующие параметры:
- `BOT_TOKEN` - токен бота от BotFather (обязательно)
- `ADMIN_ID` - Telegram ID администратора для доступа к админ-панели
