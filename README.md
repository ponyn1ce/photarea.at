# Проект «Photarea» — Создание фотокниг 📚✨

Это локальный репозиторий веб-приложения для создания фотокниг и макетов. Проект состоит из статической страницы маркетинга и встроенного редактора (React + Fabric.js), где пользователь может собирать развороты, размещать фотографии и текст, а затем сохранить или экспортировать проект.

---

## Что находится на сайте 🌐
- Главная страница (`index.html`) — промо-блоки, краткое описание продукта, FAQ и кнопки перехода в конструктор.
- Страница редактора (`editor-react/`) — полноценный WYSIWYG-редактор разворотов фотокниги.
- Статические ассеты: `css/`, `images/`, `js/` и `lang/` — стили, картинки, скрипты и переводные файлы.

---

## Что умеет редактор (кратко) ✏️🖼️
- Рендер страниц и разворотов с учётом обложки и содержимого.
- Добавление изображений (через file input и IndexedDB), их масштабирование, перемещение и применение шаблонов (center, full, half, third, quarter, collage).
- Добавление и редактирование текста (IText), смена цвета текста.
- Поддержка "спредов" (spreads) — навигация по разворотам, добавление/удаление разворотов.
- Возможность сохранять отдельные развороты в IndexedDB и метаданные в localStorage.
- Экспорт текущего разворота или всего проекта в PDF (jsPDF + svg2pdf.js).
- Локальное хранение картинок в IndexedDB (ключи, blob storage).
- Ограничение редактирования определённых страниц (например, специальные заблокированные области), визуальные подсказки и оверлеи.
- Панель миниатюр (thumbnails) для навигации и простого перетаскивания разворотов (частично реализована).
- Утилиты: очистка, удаление слоя, истории (undo stack), автосохранение и dev-кнопка для удаления всех сохранений.

## Технологии и библиотеки 🧰
- React — интерфейс редактора (`editor-react/src`).
- fabric.js — работа с canvas, объекты, клиппинг, экспорт и взаимодействие с холстом.
- jsPDF — экспорт в PDF.
- svg2pdf.js — конвертация SVG-текста в PDF (для векторного текста в PDF).
- sql.js — хранение экспортов (PDF) в локальном WebAssembly-DB (опционально).
- IndexedDB (через стандартную API) — хранение изображений и данных разворотов.
- Простые утилиты: `lang.js`, `faq.js`, `darklightbutton.js` для интерфейсных мелочей.

## Структура проекта 📁
- `index.html` — главная страница промо.
- `editor.html` — вспомогательные статические страницы.
- `editor-react/` — React-версия редактора:
  - `src/Editor.jsx` — основной компонент и логика редактора.
  - `public/` — статические ассеты для React-приложения.
- `css/` — стили (включая `main.css`, `editor.css`).
- `images/` — изображения и плейсхолдеры.
- `js/` — небольшие скрипты для статических страниц.
- `lang/` — JSON-файлы с переводами интерфейса.

## Как запустить (локально) ▶️
1. Убедитесь, что у вас установлен Node.js и npm.
2. Для статической проверки откройте `index.html` в браузере или запустите локальный статический сервер.
3. Для React-редактора (если хотите запустить dev-сервер):

```bash
cd editor-react
npm install
npm run dev
# или
npm start
```

---

## Полезные примеры команд и деплой 🛠️

- Быстрый локальный статический сервер (простая проверка `index.html`):

```powershell
npm install -g http-server
http-server . -p 8080
# затем открыть http://localhost:8080/index.html
```

- Запуск React-редактора (если используется Vite/CRA):

```powershell
cd editor-react
npm install
npm run dev   # локально для разработки
npm run build # собрать production-версию
```

- Деплой готовой сборки (простая инструкция):

1. Собрать проект `editor-react` командой `npm run build`.
2. Скопировать содержимое `editor-react/dist` (или `build`) на ваш сервер/хостинг.
3. Убедиться, что статические файлы отдаются корректно и `sql-wasm.wasm` находится в корне сборки, если вы используете sql.js.

## Полезные заметки для разработчика 🔧
- Шрифты находятся в `public/fonts/` (при необходимости замените placeholder-файлы на реальные `.woff2`).
- Сохранение разворотов: данные разворотов хранятся в IndexedDB, метаданные — в `localStorage` под ключом `savedSpreadsMeta`.
- Если изменяете логику разворотов, обновляйте функции `getSpreadPages` и рендер `renderPages` в `src/Editor.jsx`.
- В `index.html` добавлена фиксированная/липкая панель-меню — её стили перенесены в `css/main.css` и доступны через классы `.fixed-menu-panel`, `.fixed-menu-list`, `.fixed-menu-item`.
